# 电商出海决策手册 · 先判断，再出发

> **这不是一份出海教程，而是一套帮你判断"该不该出海"的决策框架。**
>
> 作者：余帅兵 | 智舟一跃电商咨询

---

## 为什么开源这份手册

过去两年，我见过太多电商老板因为"国内卷不动了"而匆忙出海，结果把国内的亏损放大十倍带回来。

也见过真正有实力的商家，因为缺少系统性的出海判断框架，在第一个市场就踩了合规红线，账号被封、货款冻结，元气大伤。

这套 Skill 最初是我给咨询客户做的内部决策工具。现在把它开源出来，希望能让更多人在做出海决定之前，先问自己几个关键问题。

**出海不是换个平台卖货，是在一个陌生的监管/物流/文化体系里重新做一次电商。**

## 手册里有什么

### 一、出海适配性诊断（4 维度决策树）

出海前先给自己打分。9 个子维度，0-10 分，总分决定你的出海节奏：

| 维度 | 子维度 | 核心问题 |
|------|--------|----------|
| 国内基盘 | ROI 健康度 | 国内过去 6 个月是否持续盈利？ |
| 国内基盘 | 产品力可复制性 | 产品在海外有没有真实差异化？ |
| 国内基盘 | 团队迁移能力 | 团队是否具备"换规则还能打"的适应力？ |
| 资源存量 | 现金流安全垫 | 能承受海外 12 个月不回本吗？ |
| 资源存量 | 供应链柔性 | 供应链能配合"小批量、多批次、短交期"吗？ |
| 资源存量 | 资金占用承受力 | 账上准备了 3 个月海外销售额的备用金吗？ |
| 风控能力 | 合规知识储备 | 清楚目标市场的 VAT/认证/知识产权红线吗？ |
| 风控能力 | 账户与资金安全 | 做过账户隔离和资金分散吗？ |
| 风控能力 | 舆情与售后承压 | 有海外客服团队吗？能 24 小时当地语言响应吗？ |

**评分规则：**
- 总分 ≥ 80：可以系统性出海，选 2 个重点市场深耕
- 总分 60-80：可以试水，但必须补短板，从单一平台切入
- 总分 40-60：不建议现在出海，先回国内补基础
- 总分 < 40：坚决不出海，出海只会让现有问题放大 10 倍

### 二、平台选择矩阵

7 大主流平台定位对照 + 4 种典型选择路径：

- **品牌型商家** → 亚马逊 + 独立站/Shopify
- **内容型商家** → TikTok Shop
- **供应链型商家** → Temu + SHEIN + 速卖通
- **新兴市场商家** → 速卖通 + TikTok Shop 东南亚

### 三、风控保命清单

四大核心风险域 + 出海前必做的"三本账"：

1. **合规账** — 税务、认证、商标、数据合规清单与成本
2. **现金流账** — 12 个月预估投入 + 止损线
3. **风险预案账** — 账户被封、资金冻结、供应链断货、舆情爆发的应对路径

> 一次严重的风控事故，就能让一家千万年 GMV 的商家彻底退出。

### 四、落地路径与退出机制

- MVP 优先：1 个平台 + 1 个市场 + 3-5 个 SKU 先跑通
- 季度复盘框架：经营指标 + 风险指标 + 战略指标
- 加码信号 vs 退出信号清单

## 如何使用

### 方式一：当作决策手册阅读（5 分钟上手）

不需要任何技术背景，直接打开 [`cross-border-skill.md`](./cross-border-skill.md) 文件，按顺序往下读。

**阅读建议：**
1. 先看「三个最危险的出海幻觉」——确认自己不是在被幻觉驱动
2. 再做「4 维度诊断」——遇到"自问清单"的地方，停下来，拿纸笔诚实打分
3. 最后看「风控保命清单」——无论诊断结果如何，这部分都值得通读

**读完之后你会得到：**
- 一个明确的出海/不出海决策
- 如果出海：知道先从哪个平台、哪个市场切入
- 如果不出海：清楚自己需要补哪些短板

### 方式二：导入 AI 助手（10 分钟设置，长期受益）

如果你使用 Claude、ChatGPT、Gemini 或国内的 Kimi、文心一言等 AI 助手，可以把这份手册导入作为定制指令。**这样以后每次聊出海话题，AI 都会自动调用这套框架，而不是泛泛而谈。**

#### 导入 Claude（推荐）

**步骤 1：创建 Project**
1. 打开 Claude 网页版：https://claude.ai
2. 左侧边栏点击 **Projects**
3. 点击 **Create Project**
4. 项目名称填：`出海决策诊断`
5. 点击 **Create Project**

**步骤 2：粘贴指令**
1. 进入刚创建的 Project
2. 点击右上角的 **Project Settings**（齿轮图标）
3. 找到 **Custom Instructions** 文本框
4. 打开本仓库的 [`cross-border-skill.md`](./cross-border-skill.md) 文件
5. 复制全部内容，粘贴到 Custom Instructions 文本框
6. 点击 **Save**

**步骤 3：开始使用**
1. 在 Project 里新建对话
2. 直接输入："我想出海，帮我看看我的情况"
3. Claude 会自动引导你做 4 维度诊断，而不是直接给操作建议

> 💡 **提示**：也可以把 `cross-border-skill.md` 作为单次对话的附件上传，效果类似，但 Project 方式可以长期保留这套指令。

---

#### 导入 ChatGPT（GPT-4 / GPT-4o）

**步骤 1：创建自定义 GPT**
1. 打开 ChatGPT 网页版：https://chat.openai.com
2. 左侧边栏点击 **Explore GPTs**
3. 点击右上角 **Create**

**步骤 2：配置指令**
1. 在 **Instructions** 标签页的文本框中，粘贴 `cross-border-skill.md` 的全部内容
2. **Name** 填：`出海决策助手`
3. **Description** 填：`帮你判断该不该出海的决策框架`
4. 点击右上角 **Save** → 选择 **Only me**

**步骤 3：开始使用**
1. 新建对话，选择你的 `出海决策助手`
2. 输入你的出海想法和现状
3. GPT 会按照 4 维度诊断框架引导你

---

#### 导入 Kimi / 文心一言 / 其他国产 AI

**通用方法（适用于所有支持"提示词"功能的 AI）：**

1. 打开 AI 助手的设置或提示词管理页面
2. 新建一个提示词/角色，名称叫：`出海决策诊断师`
3. 把 `cross-border-skill.md` 的全部内容粘贴到提示词文本框
4. 保存并启用这个提示词

**如果 AI 不支持自定义提示词：**

每次对话时，先把以下内容复制粘贴作为对话开头：

```
你是一位电商出海战略顾问。请严格按照以下框架回答我的出海相关问题：
[此处粘贴 cross-border-skill.md 的全部内容]
```

## 使用原则（使用本手册前必读）

1. **永远先劝退，再指路** — 如果不适合出海，直接说"不建议出海"
2. **用 4 维诊断主导对话** — 不要急着讲平台，先引导自测
3. **保命逻辑优先于暴富逻辑** — 先问能承受多少损失
4. **不鼓吹、不贬低任何平台** — 帮客户匹配，而不是推销

## 常见问题

**Q：这份手册能直接帮我选品或投流吗？**

不能。这份手册解决的是"战略判断"问题（该不该出、去哪里、怎么控制风险），不是"战术执行"问题（选什么品、怎么投广告）。战术层面需要结合具体平台和数据工具。

**Q：我已经在亚马逊/TikTok 上卖货了，还需要看这份手册吗？**

建议看「风控保命清单」和「退出机制」部分。很多已经在做的商家反而最容易忽视风险预警，因为"已经上了船"会产生安全幻觉。

**Q：手册里的平台信息会过时吗？**

平台规则变化很快，这份手册里的平台定位逻辑（货架电商 vs 内容电商 vs 供应链平台）是结构性判断，不容易过时。但具体的费用结构、入驻流程等细节，建议结合平台官方最新政策核实。

**Q：我可以修改这份手册给团队用吗？**

可以。MIT 开源协议允许你自由使用、修改、分发，只需保留原作者署名。
