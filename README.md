# 电商出海决策手册 · 先判断，再出发

> **这不是一份出海教程，而是一套帮你判断"该不该出海"的决策框架。**
>
> 作者：余帅兵 | 智舟一跃电商咨询

---

## 为什么开源这份手册

过去两年，我见过太多电商老板因为"国内卷不动了"而匆忙出海，结果把国内的亏损放大十倍带回来。

也见过真正有实力的商家，因为缺少系统性的出海判断框架，在第一个市场就踩了合规红线，账号被封、货款冻结，元气大伤。

这套 Skill 最初是我给咨询客户做的内部决策工具。现在把它开源出来，希望能让更多人在做出海决定之前，先问自己几个关键问题。

**出海不是换个平台卖货，是在一个陌生的监管/物流/文化体系里重新做一次电商。**

---

## 这份手册适合谁

- 国内电商做了几年，感觉增长见顶，在犹豫要不要出海的老板
- 已经出了海但踩了坑，想复盘自己当初决策哪里出了问题的创业者
- 给电商客户做战略咨询的同行，想借用一个结构化框架

**不适合谁：**

- 想找"如何在 TikTok 上爆单"操作教程的人——这不是操作指南
- 只想听好消息、不想被泼冷水的人——这份手册的第一原则是"先劝退，再指路"

## 核心内容

### 一、出海适配性诊断（4 维度决策树）

出海前先给自己打分。9 个子维度，0-10 分，总分决定你的出海节奏：

| 维度 | 子维度 | 核心问题 |
|------|--------|----------|
| 国内基盘 | ROI 健康度 | 国内过去 6 个月是否持续盈利？ |
| 国内基盘 | 产品力可复制性 | 产品在海外有没有真实差异化？ |
| 国内基盘 | 团队迁移能力 | 团队是否具备"换规则还能打"的适应力？ |
| 资源存量 | 现金流安全垫 | 能承受海外 12 个月不回本吗？ |
| 资源存量 | 供应链柔性 | 供应链能配合"小批量、多批次、短交期"吗？ |
| 资源存量 | 资金占用承受力 | 账上准备了 3 个月海外销售额的备用金吗？ |
| 风控能力 | 合规知识储备 | 清楚目标市场的 VAT/认证/知识产权红线吗？ |
| 风控能力 | 账户与资金安全 | 做过账户隔离和资金分散吗？ |
| 风控能力 | 舆情与售后承压 | 有海外客服团队吗？能 24 小时当地语言响应吗？ |

**评分规则：**
- 总分 ≥ 80：可以系统性出海，选 2 个重点市场深耕
- 总分 60-80：可以试水，但必须补短板，从单一平台切入
- 总分 40-60：不建议现在出海，先回国内补基础
- 总分 < 40：坚决不出海，出海只会让现有问题放大 10 倍

### 二、平台选择矩阵

7 大主流平台定位对照 + 4 种典型选择路径：

- **品牌型商家** → 亚马逊 + 独立站/Shopify
- **内容型商家** → TikTok Shop
- **供应链型商家** → Temu + SHEIN + 速卖通
- **新兴市场商家** → 速卖通 + TikTok Shop 东南亚

### 三、风控保命清单

四大核心风险域 + 出海前必做的"三本账"：

1. **合规账** — 税务、认证、商标、数据合规清单与成本
2. **现金流账** — 12 个月预估投入 + 止损线
3. **风险预案账** — 账户被封、资金冻结、供应链断货、舆情爆发的应对路径

> 一次严重的风控事故，就能让一家千万年 GMV 的商家彻底退出。

### 四、落地路径与退出机制

- MVP 优先：1 个平台 + 1 个市场 + 3-5 个 SKU 先跑通
- 季度复盘框架：经营指标 + 风险指标 + 战略指标
- 加码信号 vs 退出信号清单

## 如何使用

### 方式一：当作决策手册阅读

直接打开 `cross-border-skill.md`，按顺序读。遇到"自问清单"的地方，停下来，诚实回答。

### 方式二：在 AI 助手里使用

如果你使用 Claude、ChatGPT、Gemini 等 AI 助手，可以把 `cross-border-skill.md` 的内容作为 System Prompt 或 Custom Instruction 导入。

**效果：** 当你问"我想出海，你觉得怎么样"时，AI 会先引导你做 4 维诊断，而不是直接给你平台操作建议。

**导入方法（以 Claude 为例）：**
1. 打开 Claude → Projects → 新建 Project
2. 在 Project Instructions 中粘贴 `cross-border-skill.md` 的内容
3. 每次在这个 Project 里聊出海话题，AI 都会自动调用这套框架

---

## 使用原则

1. **永远先劝退，再指路** — 如果不适合出海，直接说"不建议出海"
2. **用 4 维诊断主导对话** — 不要急着讲平台，先引导自测
3. **保命逻辑优先于暴富逻辑** — 先问能承受多少损失
4. **不鼓吹、不贬低任何平台** — 帮客户匹配，而不是推销
